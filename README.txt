Untitled Game folderyje yra sukurtas žaidimas, klonuokite saugyklą ir paleiskite exe failą norėdami pradėti žaidimą. Norint tęstuoti dviejų žaidėjų rėžimą, du kartus atsidarykite exe failą ir sukurkite per vieną serverį, per kitą bandykite prisijungti. 

Scripts folderyje pateikti pagrindiniai bakalauro darbe apžvelgti programiniai kodai.
